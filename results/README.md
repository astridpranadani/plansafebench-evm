# Results directory

This directory contains reviewer-facing experimental artifacts for PlanSafeBench-EVM.

Included result groups:

- `main_v5_6/`: corrected main benchmark metrics and predictions under v5.6.
- `llm_audit_v5_6/`: corrected LLM-generated plan audit results under v5.6.
- `expert_validation_v0_4/`: public expert-validation summary artifacts. Raw expert response files and reviewer-identifying information are not included.
- `counterfactual_v5_6_harmonized/`: harmonized counterfactual robustness results.
- `v0_3_hardening/`: selected supporting summary tables for bootstrap confidence intervals, McNemar tests, simple rule-baseline summaries, prompt-variant summaries, and violation-frequency analysis.

These result tables are based on benchmark reference decisions and encoded semantic-policy labels. They should not be described as independent human ground truth.
