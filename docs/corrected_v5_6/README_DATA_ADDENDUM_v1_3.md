# PlanSafeBench-EVM Repository Data Addendum v1.3

This addendum contains the raw/required files referenced by the repository README for full reproduction:

- `data/raw/transaction_templates/`: Level-2 real Ethereum transaction templates.
- `data/raw/benchmark_scenarios/`: original v4.8 benchmark scenario files and split file.
- `data/raw/llm_prompts/`: LLM-generated plan audit prompts and model manifest template.
- `data/llm_outputs/`: raw LLM outputs for Gemini, Qwen, and Claude Haiku full 1,760-prompt runs.

Copy these files into the root of the GitHub repository before tagging the release.
